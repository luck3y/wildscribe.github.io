  4. address-setting

An address setting defines some attributes that are defined against an address wildcard rather than a specific queue.

Attributes (12)

  • address-full-policy Determines what happens when an address where max-size-bytes is specified becomes full. (PAGE, DROP or BLOCK)
    • Attribute Value
    Default Value PAGE
    Type STRING
    Nillable true
    Expressions Allowed true
  • dead-letter-address The dead letter address
    • Attribute Value
    Type STRING
    Nillable true
    Expressions Allowed true
  • expiry-address Defines where to send a message that has expired.
    • Attribute Value
    Type STRING
    Nillable true
    Expressions Allowed true
  • last-value-queue Defines whether a queue only uses last values or not
    • Attribute Value
    Default Value false
    Type BOOLEAN
    Nillable true
    Expressions Allowed true
  • max-delivery-attempts Defines how many time a cancelled message can be redelivered before sending to the dead-letter-address
    • Attribute Value
    Default Value 10
    Type INT
    Nillable true
    Expressions Allowed true
  • max-size-bytes The max bytes size.
    • Attribute Value
    Default Value -1
    Type LONG
    Nillable true
    Expressions Allowed true
  • message-counter-history-day-limit Day limit for the message counter history.
    • Attribute Value
    Default Value 0
    Type INT
    Nillable true
    Expressions Allowed true
  • page-max-cache-size The number of page files to keep in memory to optimize IO during paging navigation.
    • Attribute Value
    Default Value 5
    Type INT
    Nillable true
    Expressions Allowed true
  • page-size-bytes The paging size.
    • Attribute Value
    Default Value 10485760
    Type LONG
    Nillable true
    Expressions Allowed true
  • redelivery-delay Defines how long to wait before attempting redelivery of a cancelled message
    • Attribute Value
    Default Value 0
    Type LONG
    Nillable true
    Expressions Allowed true
  • redistribution-delay Defines how long to wait when the last consumer is closed on a queue before redistributing any messages
    • Attribute Value
    Default Value -1
    Type LONG
    Nillable true
    Expressions Allowed true
  • send-to-dla-on-no-route If this parameter is set to true for that address, if the message is not routed to any queues it will instead be sent to the dead letter address (DLA) for that address, if it exists.
    • Attribute Value
    Default Value false
    Type BOOLEAN
    Nillable true
    Expressions Allowed true
